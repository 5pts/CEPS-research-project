1. 项目概述

1.1 研究背景与目标

本项目旨在探究“学校社会资本”对初中生教育期望的影响，特别是其在不同家庭背景（SES、户籍）学生中的“补偿效应”。核心问题是：学校教育资源能否帮助弱势家庭学生打破阶层固化，提升升学信心？

1.2 上阶段工作回顾

4. 存在问题与解决方案

**问题 1：原始变量选择导致样本雪崩**

原因：原计划使用的“最高学历期望”题目属于跳答题，仅非普高意愿者回答，导致80%缺失。

解决：已替换为全员必答的“期望学历”题目，并重新界定了二分变量，彻底解决了此问题。

**问题 2：单层模型无法处理嵌套结构**

困难：学生嵌套于班级和学校，普通回归标准误不准。

解决：已整合学校层级数据 (Level-2)，为下一步实施 HLM (多层线性模型) 做好了数据准备。
